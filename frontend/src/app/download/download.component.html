Download results

On this page, you can download the results of the survey. The data are password protected. Enter the password and select which table you want to download.

There are three tables for the results: an overview of works, heroes, or responses. Some notes:

  • The tables of works and heroes respectively list all works or heroes that have been submitted in the form, including ones for which there are no personal responses.
  • The data is provided as a .tsv file. You should be able to import this in editors like Excel or SPSS without issue.
